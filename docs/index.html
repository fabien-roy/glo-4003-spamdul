/api/users post

post

Add a new user

post /api/users

Add a new user

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • name: required(string)

    Full name of user

  • birthDate: required(string)

    Birth date of the user

  • sex: required(one of m, f, x)

    Valid sex

  • accessDay: required(one of monday, tuesday, wednesday, thrusday, friday, saturday, sunday)

    Day of access to parking

Example:

{
  "name": "John Doe",
  "birthDate": "01-01-2000",
  "sex": "m",
  "accessDay": "monday"
}

/api/parkings post

post

Add a new parking sticker

post /api/parkings

Add a new parking sticker

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • accountId: required(string)

    UUID representing account

  • parkingArea: required(string)

    Code representing parking area

  • receptionMethod: required(one of postal, email)

    Valid reception method

  • postalCode: (string)

    Canadian postal code to send sticker to, required if reception method is postal

  • email: (string)

    Email to send sticker to, required if reception method is email

  • validDay: required(one of monday, tuesday, wednesday, thrusday, friday, saturday, sunday)

    Day of sticker validity

Examples:

postal:

{
  "accountId": "9af7ec3f-7383-4280-9e75-31f32933347a",
  "parkingArea": "Zone1",
  "receptionMethod": "postal",
  "postalCode": "A1E 2E1",
  "validDay": "monday"
}

email:

{
  "accountId": "9af7ec3f-7383-4280-9e75-31f32933347a",
  "parkingArea": "Zone1",
  "receptionMethod": "email",
  "email": "spamd.ul@ulaval.ca",
  "validDay": "monday"
}

HTTP status code 201

Code for newly added parking sticker

Body

Media type: application/json

Type: object

Properties
  • parkingStickerCode: required(string)

    Code representing parking area

Example:

{
  "parkingStickerCode": "SPAVUL-123"
}

HTTP status code 400

Error adding parking sticker

HTTP status code 404

Account or parking area not found
